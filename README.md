# Ian Marcus Q. Breganza — Portfolio

Personal portfolio website for Ian Marcus Q. Breganza, CS Graduate & ML Engineer.

## Tech Stack

- **HTML5** — semantic structure
- **Tailwind CSS** (CDN) — utility-first styling
- **Vanilla JavaScript** — interactivity and dynamic content
- **Formspree** — contact form backend

## Features

- Animated hero with floating ambient orbs and a reactive gradient on the name
- Scroll-reveal animations on all sections
- Expandable origin story timeline
- Dynamically rendered Projects, Experience, Skills, and Certificates sections
- Scrollable content areas with edge-fade masks (Projects & Experience)
- Formspree-powered contact form
- Mobile-responsive layout with hamburger menu
- Custom noise grain overlay and dark theme

## Structure

```
frontend/
├── index.html        # Main page
├── css/
│   └── style.css     # Custom styles and animations
├── js/
│   └── app.js        # Data, rendering logic, and animation sync
└── Breganza_grad11.png
```

## Running Locally

Open `index.html` directly in a browser — no build step required.

## Deployment

Hosted via GitHub Pages. Push to `main` to trigger a rebuild.
